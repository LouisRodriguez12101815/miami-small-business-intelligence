## For participants who want to run it locally

```bash
git clone https://github.com/LouisRodriguez12101815/miami-small-business-intelligence.git
cd miami-small-business-intelligence
npm install
npm run dev
# Open http://localhost:5173
```

## For participants who want their own deploy

Anyone can fork the repo and deploy to their own Vercel account:

1. Fork https://github.com/LouisRodriguez12101815/miami-small-business-intelligence
2. Sign up at https://vercel.com (free)
3. Import the forked repo → Vercel auto-detects Vite
4. Deploy — no env variables needed
